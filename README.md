# Proyecto-de-analisis-de-datos-sobre-fraude-con-tarjetas-de-credito
# Análisis de reclamaciones por fraude con tarjetas.

## Resumen del Proyecto

Este proyecto analiza las reclamaciones de consumidores relacionadas con fraude con tarjetas de crédito utilizando una muestra de 5000 registros de la Base de Datos de Quejas de Consumidores. El objetivo del análisis es identificar patrones en las reclamaciones relacionadas con fraude, determinar qué instituciones financieras reciben más quejas y analizar la evolución de estos incidentes con el tiempo.

## Conjunto de Datos

El conjunto de datos proviene de la Base de Datos de Quejas de Consumidores e incluye quejas presentadas por consumidores sobre productos y servicios financieros.

Para este proyecto, el conjunto de datos se redujo a 5000 registros para facilitar el análisis con Hojas de Cálculo de Google.

## Herramientas Utilizadas

- Hojas de Cálculo de Google
- Tablas Dinámicas
- Limpieza de Datos
- Visualización de Datos
- Comandos de Terminal

## Hallazgos Clave

- La queja más común se refiere a cargos no autorizados en cuentas.
- Citibank aparece como la entidad con el mayor número de reclamaciones en el conjunto de datos, en este caso se observó un pico de quejas en **abril de 2019**.
